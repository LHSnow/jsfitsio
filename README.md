# JSFITSIO

Web assembly (wasm) wrapper of [CFITSIO](https://heasarc.gsfc.nasa.gov/docs/software/fitsio/fitsio.html), 
to interact with astronomical and atmospherical scientific imagery of FITS files in a browser setting.  

## Build

```bash
emmake make all
```

or

```bash
npm run build
```
